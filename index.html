¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, ligero y multiparadigma, diseñado para crear páginas web interactivas. Se ejecuta directamente en el navegador, permitiendo que los sitios respondan a acciones del usuario sin recargar la página.

  • Lenguaje interpretado: No necesita compilación previa.
  • Multiparadigma: Soporta programación orientada a objetos, funcional y estructurada.
  • Ejecutado en el cliente: Corre en el navegador del usuario.
  • Aplicaciones comunes: Validación de formularios, animaciones, juegos, interacciones dinámicas.

Conceptos Fundamentales

  • Variables: let, const, var
  • Tipos de datos: String, Number, Boolean, Array, Object
  • Funciones: Tradicionales y flecha (() => {})
  • Condicionales: if, else, switch
  • Bucles: for, while, forEach
  • Eventos: onclick, onchange, onmouseover
  • Manipulación del DOM: document.querySelector, innerHTML
  • Promesas y async/await: Para manejar operaciones asincrónicas
